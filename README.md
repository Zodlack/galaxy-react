# Galaxy — Силомер

Тестовое задание, реализованное на React + Vite + CSS Modules.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Что реализовано

- стартовый экран;
- экран ожидания удара;
- хаотично меняющаяся шкала силы слева;
- анимация удара молота;
- динамическое заполнение основной шкалы по результату удара;
- экран поражения;
- экран победы;
- повторный старт игры.

## Примечания

- игровая область зафиксирована в размере `360x640`;
- стили разбиты на CSS Modules;
- логика игры хранится в локальном состоянии React;
- ассеты из рабочего архива включены только те, которые используются в приложении.
